Kenyan Restaurant Website

A modern Afro-luxury restaurant website designed to showcase authentic Kenyan cuisine through immersive visuals, smooth animations, and a premium user experience.

This project combines modern web design with Kenyan culture, featuring beautifully presented local dishes, elegant layouts, interactive transitions, and a fully responsive mobile-first experience.

Features
Smooth scrolling animations
Interactive food menu
Premium Afro-modern UI design
Fully responsive layout
Reservation section
WhatsApp integration
Food gallery with hover effects
Modern navigation and transitions
Kenyan Shilling (KSh) pricing
Optimized for mobile users
